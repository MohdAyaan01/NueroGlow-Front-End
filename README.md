## 🚀 Overview
**AiSaaS** is a multi-purpose AI platform that brings state-of-the-art machine learning capabilities to your fingertips. From generating high-quality articles to professional-grade image editing (background/object removal), AiSaaS is built to be the one-stop solution for creators, developers, and businesses.

## ✨ Key Features
- ✍️ **AI Content Generation**: Create long-form articles and catchy blog titles in seconds.
- 🖼️ **Advanced Image Editing**:
  - **Background Removal**: Clean, high-precision background extraction.
  - **Object Removal**: Seamlessly erase unwanted elements from images.
- 🔐 **Secure Authentication**: Integrated Google OAuth and secure custom auth flows.
- 💳 **Monetization Ready**: Full **Razorpay** integration for subscription and credit management.
- 👥 **Community Hub**: Share your creations and explore works from other users.
- 🎨 **Modern Interface**: A sleek, dark-themed UI built with **Tailwind CSS 4** and **Framer Motion**.

## 🛠️ Tech Stack
- **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/) & [Redux Persist](https://github.com/rt2zz/redux-persist)
- **Routing**: [React Router Dom v7](https://reactrouter.com/)
- **Payments**: [Razorpay SDK](https://razorpay.com/docs/payments/payment-gateway/web-integration/standard/)
- **Authentication**: [Google OAuth](https://developers.google.com/identity/gsi/web/guides/overview)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🏗️ Getting Started

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/MohdAyaan01/AiSaaS.git
   cd AiSaaS/client
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file:
   ```env
   VITE_API_URL=http://localhost:5000
   VITE_GOOGLE_CLIENT_ID=your_google_id
   VITE_RAZORPAY_KEY_ID=your_razorpay_id
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```
